# 프로젝트 주제

IT 기술을 활용하여 현대인의 건강 (wellness) 문제를 해결할 수 있는 웹사이트 구축 -> 컴퓨터를 주로 하는 사람들의 삶의 질을 올려줄 수 있는 힐링 비서

# 제작 기간
2024.07.05 ~ 2024.08.06

# 참여 인원
프론트엔드 - 3명
<br/>
백엔드 - 2명

# 사용한 기술 (기술 스택)

프론트엔드 - HTML , CSS , JS , REACT
백엔드 - SPRINGBOOT , SPRINGSECURITY , MYSQL

# 핵심 기능

- 배경화면 전환
- ASMR 기능
- 캘린더 기능
- TODOLIST 기능
- 유튜브 API를 활용한 음악 플레이어
